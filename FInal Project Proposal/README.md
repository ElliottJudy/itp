##Paragraph Overview:
This will be a 2d platformer game with a shooter game mechanic made in gamemaker. You will be able to run and jump like a platforming game while also having the ability to aim and shoot projectiles with the mouse cursor. There will be a tilte screen and music with the game. I will also make some very basic pixel art for the game. The game will have at least 1 level. 

##Overlap:
The only thing this project will overlap with is that I have worked with gamemaker a little bit in the past. I have wanted to make a game like this and this will give me a start while also having a small amount of background knowledge.

##Outcomes:

###Good Outcome:
A good outcome would be a game which functionally has all the features I listed above like working platforming and shooting mechanics, a title screen, and 1 level but its lacking in the way it looks and the game is not much fun to play yet due to poor design

###Better Outcome:
A better outcome would be all the things I mentioned plus a few more levels, better looking graphics, and be somewhat fun to play.

###Best Outcome:
The best outcome would be a complete game which has an ending, is fun to play, and has aestheticly pleasing graphics.
